# Curated Events

A collection of seven `.ics` calendar files covering religious, federal, cultural, civic, and honorary observances from 2026 through 2036, suitable for import into any calendar application that supports the iCalendar format.

## Calendars

### christian.ics

Key Protestant liturgical and observance dates. The following events are included for each year in the range:

| Event                       | Description                                                                                      |
| --------------------------- | ------------------------------------------------------------------------------------------------ |
| **Ascension Day**           | Commemorates Jesus' ascension into heaven 40 days after his resurrection                         |
| **First Day of Lent**       | Begins the 40-day season of fasting, prayer, and repentance leading to Resurrection Sunday       |
| **First Sunday of Advent**  | Marks the beginning of the four-week season of preparation leading up to Christmas               |
| **Good Friday**             | Commemorates the crucifixion and death of Jesus Christ at Calvary                                |
| **National Day of Prayer**  | First Thursday of May — calls people of faith to unite in prayer for the nation                  |
| **Palm Sunday**             | Commemorates Jesus' triumphal entry into Jerusalem; one week before Resurrection Sunday          |
| **Pastor Appreciation Day** | Second Sunday of October — honors pastors and spiritual leaders                                  |
| **Pentecost**               | Commemorates the descent of the Holy Spirit upon the apostles, 50 days after Resurrection Sunday |
| **Reformation Day**         | October 31 — commemorates Martin Luther's posting of his 95 Theses in 1517                       |
| **Resurrection Sunday**     | The central Christian celebration of Jesus Christ's bodily resurrection                          |

10 events × 11 years = **110 total events**

Use the "Add calendar from URL" or "Subscribe" feature in your app and paste the desired URL.

### Import into a Calendar App

1. Download the desired `.ics` file or use the URL above
2. Open your calendar application (Google Calendar, Apple Calendar, Proton Calendar, Outlook, etc.)
3. Use the import or "Add calendar from file" feature to load the `.ics` file

### Proton Calendar Notes

Proton Calendar has the following limitations to be aware of:

- Maximum file size: 10 MB
- Maximum events per import: 15,000
- Date range support appears to cap around 12–13 years out

All files are sized well within those constraints.

## Format

The files follow the [iCalendar (RFC 5545)](https://datatracker.ietf.org/doc/html/rfc5545) standard:

- `CALSCALE:GREGORIAN`
- All events are all-day (`VALUE=DATE`)
- Timezone reference: `America/Chicago`
- Each event has a unique `UID` for deduplication on re-import
